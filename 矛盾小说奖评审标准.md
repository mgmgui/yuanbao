# 矛盾小说奖评审标准

## 评审目标
确保小说达到矛盾小说奖标准，具有深刻的人物矛盾展现和文学质量。

## 评审维度（每个维度20分，总分100分）

### 维度一：矛盾深度（20分）
**评分标准**：
1. **人物内心矛盾的真实性**（5分）：人物内心挣扎的真实性和复杂性
2. **矛盾展现的层次性**（5分）：矛盾展现的层次感和深度
3. **矛盾对情节的影响**（5分）：矛盾如何推动情节发展
4. **矛盾解决的艺术性**（5分）：矛盾如何被艺术性解决或未解决

**评分细则**：
- 20分：矛盾极其深刻，贯穿整个故事，影响人物命运
- 16-19分：矛盾深刻，展现层次感，推动情节发展
- 12-15分：矛盾有一定展现，但不够深刻
- 8-11分：矛盾展现肤浅，不够复杂
- 0-7分：矛盾不明显或缺失

### 维度二：文学质量（20分）
**评分标准**：
1. **语言艺术性**（5分）：语言优美，表达准确，富有文学性
2. **结构完整性**（5分）：结构完整，章节安排合理，逻辑清晰
3. **情节合理性**（5分）：情节发展合理，符合逻辑
4. **主题深刻性**（5分）：主题深刻，思想性强

**评分细则**：
- 20分：语言极其优美，结构极其完整，主题极其深刻
- 16-19分：语言优美，结构完整，主题深刻
- 12-15分：语言合格，结构合理，主题有一定深度
- 8-11分：语言平淡，结构混乱，主题肤浅
- 0-7分：语言低劣，结构缺失，主题不明

### 维度三：人物塑造（20分）
**评分标准**：
1. **人物真实性**（5分）：人物形象真实，符合逻辑
2. **人物复杂性**（5分）：人物复杂，具有多重性格特点
3. **人物发展性**（5分）：人物发展符合逻辑，有成长过程
4. **人物关系性**（5分）：人物关系合理，有互动性

**评分细则**：
- 20分：人物极其真实，极其复杂，极其发展，关系极其合理
- 16-19分：人物真实，复杂，发展，关系合理
- 12-15分：人物基本真实，有一定复杂性和发展性
- 8-11分：人物不够真实，不够复杂，不够发展
- 0-7分：人物虚假，单一，不变，关系混乱

### 维度四：情节设计（20分）
**评分标准**：
1. **情节冲突性**（5分）：情节冲突强烈，戏剧性强
2. **情节逻辑性**（5分）：情节逻辑清晰，符合逻辑
3. **情节新颖性**（5分）：情节新颖，富有创意
4. **情节完整性**（5分）：情节完整，结构清晰

**评分细则**：
- 20分：情节冲突极其强烈，极其逻辑，极其新颖，极其完整
- 16-19分：情节冲突强烈，逻辑清晰，新颖，完整
- 12-15分：情节有一定冲突，逻辑基本清晰，有一定新颖性
- 8-11分：情节冲突不足，逻辑不够清晰，不够新颖
- 0-7分：情节冲突缺失，逻辑混乱，陈旧，不完整

### 维度五：历史真实性（20分）
**评分标准**：
1. **历史背景真实性**（5分）：历史背景真实，符合历史事实
2. **文化细节准确性**（5分）：文化细节准确，符合历史文化
3. **时代特征真实性**（5分）：时代特征真实，符合时代特点
4. **历史逻辑合理性**（5分）：历史逻辑合理，符合历史逻辑

**评分细则**：
- 20分：历史背景极其真实，文化细节极其准确，时代特征极其真实，历史逻辑极其合理
- 16-19分：历史背景真实，文化细节准确，时代特征真实，历史逻辑合理
- 12-15分：历史背景基本真实，文化细节基本准确，时代特征基本真实
- 8-11分：历史背景不够真实，文化细节不够准确，时代特征不够真实
- 0-7分：历史背景虚假，文化细节错误，时代特征缺失

## 矛盾小说奖标准

### 矛盾小说奖核心标准
**矛盾深度必须达到以下标准**：
1. 人物内心矛盾展现深刻，真实反映内心挣扎
2. 矛盾具有多重层次，展现复杂性
3. 矛盾影响人物命运，推动情节发展
4. 矛盾具有艺术性解决或不解决的安排

### 矛盾小说奖合格标准
**总分必须≥90分，且矛盾深度≥18分**：
- 总分≥90分：文学质量达标
- 矛盾深度≥18分：矛盾展现深刻达标
- 人物塑造≥16分：人物塑造达标
- 情节设计≥16分：情节设计达标
- 历史真实性≥16分：历史真实性达标

## 评审结果处理

### 评审达标情况处理
**达标**（总分≥90分且矛盾深度≥18分）：
- 直接交付用户
- 记录评审分数和达标情况
- 归档文件

**不达标但可修改**（总分≥80分但矛盾深度＜18分）：
- 退回构思人物Agent修改人物档案
- 修改人物矛盾深度
- 重新评审

**不合格**（总分＜80分）：
- 退回构思小说Agent重写
- 重新创作小说正文
- 重新评审

### 修改要求
**退回修改时必须明确指出**：
1. 具体问题所在（哪个维度不足，哪个评分项不足）
2. 具体修改方向（如何修改，如何提升）
3. 具体修改示例（修改示例，改进方向）

**重新创作要求**：
1. 重新创作人物档案（构思人物Agent）
2. 重新创作背景构思（构思背景Agent）
3. 重新创作创意构思（构思创意Agent）
4. 重新创作小说正文（构思小说Agent）

## 评审报告格式

### 评审报告必须包含：
1. **评审基本信息**
   - 评审小说名称
   - 评审人
   - 评审时间
   - 评审分数（每个维度分数和总分）

2. **维度评分分析**
   - 矛盾深度评分分析
   - 文学质量评分分析
   - 人物塑造评分分析
   - 情节设计评分分析
   - 历史真实性评分分析

3. **优点分析**
   - 作品的亮点
   - 特别优秀的部分
   - 值得称赞的创作技巧

4. **缺点指出**
   - 存在的问题
   - 需要改进的部分
   - 不符合要求的方面

5. **改进建议**
   - 具体修改建议
   - 修改方向
   - 修改示例

6. **综合评价**
   - 总体评价
   - 是否达标
   - 处理建议

### 评审结果处理：
**达标**：
- 评级：五星
- 处理：直接交付
- 记录：记录评审分数

**不达标但可修改**：
- 评级：四星或三星
- 处理：退回修改
- 修改要求：明确指出问题并提供修改建议

**不合格**：
- 评级：二星或一星
- 处理：退回重写
- 重写要求：重新创作所有部分

### 评审报告示例
```
织田信长小说评审报告
评审时间：2026年3月25日
评审人：矛盾小说奖评审委员会

维度评分：
矛盾深度：18分（人物内心矛盾展现深刻）
文学质量：17分（语言优美，结构完整）
人物塑造：18分（人物真实，复杂，发展）
情节设计：19分（情节冲突强烈，新颖）
历史真实性：18分（历史背景真实，文化细节准确）
总分：90分

优点分析：
1. 人物内心矛盾展现深刻，真实反映内心挣扎
2. 情节冲突强烈，戏剧性强
3. 语言优美，表达准确，富有文学性
4. 历史背景真实，符合历史事实

缺点指出：
1. 人物感情线不够深刻，需要加强
2. 高潮场景戏剧性不够强烈，需要加强

改进建议：
1. 加强人物感情线的刻画，增加感情冲突
2. 增强高潮场景的戏剧性，增加冲突强度

综合评价：
总分90分，矛盾深度18分，达到矛盾小说奖标准。
评级：五星
处理建议：直接交付用户
```
