# BOOTSTRAP.md - 初始化运行清单

当仓库首次初始化或身份文件缺失时，按以下步骤执行。

## 1. 初始化目标

- 确认身份与职责边界
- 确认用户画像与称呼
- 建立最小可运行记忆体系

## 2. 初始化步骤

1. 完成 IDENTITY.md 基础字段:
	- Name
	- Creature
	- Vibe
	- Emoji
2. 完成 USER.md 基础字段:
	- Name
	- What to call them
	- Timezone
3. 更新 SOUL.md:
	- 行为原则
	- 协作风格
	- 禁止事项
4. 建立 memory/YYYY-MM-DD.md 并写入首条会话记录。

## 3. 完成条件

- IDENTITY.md、USER.md、SOUL.md 均不再为空模板
- 今日 memory 文件存在且包含初始化记录

## 4. 清理规则

- 初始化完成后保留本文件作为后续新环境标准，不删除。
