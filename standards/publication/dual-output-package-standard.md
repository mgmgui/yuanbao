# 双平台交付包标准

## 1. 包结构

每次交付必须包含:

- wechat_markdown: 微信稿
- xiaohongshu_markdown: 小红书稿
- source_version: 源小说版本
- review_report: 质量评审报告
- pre_publish_checklist: 发布前检查结果

## 2. 命名规则

- deliverables/packages/<topic>-package-vX.Y/
  - wechat.md
  - xiaohongshu.md
  - review-report.md
  - pre-publish-checklist.md
  - metadata.json

## 3. 最小可发布标准

- 双稿均通过各自格式 SOP
- 双稿均通过发布前检查
- 评审结论为 PASS
