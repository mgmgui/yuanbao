# 阶段检查清单

## 第一阶段：人物塑造Agent Checklist

### 任务要求
- 创作50个日本人物档案
- 每日产出：10个档案

### 档案质量标准检查清单
1. **基本信息完整性**
   - 姓名（日本式姓名）
   - 年龄
   - 性别
   - 出生地
   - 职业/身份

2. **背景故事深度**
   - 家庭背景
   - 教育经历
   - 职业历程
   - 关键人生事件
   - 社会关系网络

3. **性格特点多维性**
   - 主要性格特质（至少3个）
   - 性格矛盾点
   - 行为习惯
   - 价值观体系
   - 情感模式

4. **人物弧光完整性**
   - 起点状态
   - 成长轨迹
   - 转折点
   - 终点状态
   - 成长动力

5. **主题意象明确性**
   - 核心主题（如：孤独、传统与现代、家庭羁绊等）
   - 象征元素（如：樱花、茶道、武士精神等）
   - 文化隐喻
   - 哲学内涵

6. **文化准确性验证**
   - 日本文化元素准确度
   - 社会阶层合理性
   - 历史背景一致性
   - 语言表达适当性
   - 礼仪习俗准确性

### 每日交付检查
- 文件名符合规范：`人物-<序号>-<姓名>.md`
- 文件格式：Markdown
- 字数要求：500-1000字
- 内容结构完整
- 无语法错误

## 第二阶段：小说创作Agent Checklist

### 任务要求
- 基于50个人物档案创作50篇短篇小说
- 每日产出：5篇小说

### 小说质量标准检查清单
1. **字数达标**
   - 字数范围：5,000-8,000字
   - 章节分布合理

2. **结构完整性**
   - 开头：吸引读者，引出主要人物
   - 发展：情节推进，人物发展
   - 高潮：冲突解决，情感爆发
   - 结局：圆满或有意义的结束
   - 尾声：余韵或启示

3. **文学质量评估**
   - 语言流畅度
   - 描写生动性
   - 对话自然度
   - 节奏掌控
   - 情感感染力

4. **人物塑造一致性**
   - 与档案人物特征一致
   - 人物行为符合性格
   - 人物发展符合弧光
   - 人物关系合理发展

5. **主题表达清晰度**
   - 核心主题明确体现
   - 象征元素合理运用
   - 文化内涵自然融入
   - 哲学思考恰当呈现

### 每日交付检查
- 文件名符合规范：`小说-<序号>-<标题>.md`
- 文件格式：Markdown
- 章节划分清晰
- 无语法错误
- 内容连贯完整

## 第三阶段：评审Agent Checklist

### 任务要求
- 对50篇小说进行评审
- 每日产出：评审10篇小说

### 评审质量标准检查清单
1. **评分系统统一性**
   - 五星评级标准清晰
   - 评分标准一致性
   - 评分维度完整

2. **优点分析深度**
   - 至少识别3个优点
   - 优点描述具体
   - 优点分析有深度
   - 优点与作品价值关联

3. **改进建议实用性**
   - 至少提供3个改进建议
   - 建议具体可操作
   - 建议有针对性
   - 建议有建设性

4. **综合评价全面性**
   - 总体评价客观公正
   - 潜力分析合理
   - 改进方向明确
   - 未来创作建议

5. **评审格式规范性**
   - 评审报告结构完整
   - 语言专业客观
   - 引用具体段落
   - 举例说明清晰

### 每日交付检查
- 文件名符合规范：`评审-<序号>-<小说标题>.md`
- 文件格式：Markdown
- 评分表完整
- 建议具体明确
- 评审逻辑清晰

## 第四阶段：归档管理Checklist

### 任务要求
- 整理所有阶段的成果文件
- 生成统计数据和质量报告
- 撰写项目总结报告

### 归档质量标准检查清单
1. **文件完整性**
   - 所有人物档案齐全（50个）
   - 所有小说稿齐全（50个）
   - 所有评审报告齐全（50个）
   - 所有项目管理文件齐全

2. **文件组织规范性**
   - 目录结构清晰
   - 文件名符合规范
   - 文件分类正确
   - 文件存储有序

3. **统计数据准确性**
   - 进度统计数据准确
   - 质量统计数据准确
   - 时间统计数据准确
   - 效率统计数据准确

4. **项目总结完整性**
   - 项目过程总结完整
   - 成果分析深入
   - 经验教训提炼
   - 未来建议实用

5. **归档格式规范性**
   - 归档报告格式规范
   - 图表数据清晰
   - 文字表述专业
   - 文件链接正确

### 交付检查
- 归档目录结构完整
- 统计数据文件齐全
- 项目总结报告完整
- 质量控制报告详细
- 所有文件可追溯
