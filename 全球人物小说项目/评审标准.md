# 小说评审标准

## 评审目标
确保每篇小说达到矛盾标准和拿奖标准，符合出版质量要求。

## 评审维度（每个维度20分，总分100分）

### 维度一：内心挣扎（20分）
**评分标准**：
- 人物内心矛盾的真实性和深度
- 内心冲突的展现层次和复杂度
- 内心变化的连贯性和合理性
- 内心挣扎对人物命运的影响

**评分细则**：
- 0-10分：内心挣扎不明显或肤浅
- 11-15分：内心挣扎有一定展现但不够深刻
- 16-18分：内心挣扎深刻，有层次感
- 19-20分：内心挣扎极其深刻，贯穿整个故事

### 维度二：感情线（20分）
**评分标准**：
- 爱情/亲情/友情的发展合理性
- 情感线的逻辑性和连贯性
- 情感冲突的戏剧性和真实性
- 感情发展对情节的影响

**评分细则**：
- 0-10分：感情线混乱或不合理
- 11-15分：感情线基本合理但不深刻
- 16-18分：感情线合理且有一定深度
- 19-20分：感情线极其深刻，推动情节发展

### 维度三：高潮场景（20分）
**评分标准**：
- 关键时刻的戏剧性和震撼力
- 高潮场景的铺垫和爆发节奏
- 高潮场景的情绪冲击力
- 高潮场景对主题的升华

**评分细则**：
- 0-10分：高潮场景平淡或不合理
- 11-15分：高潮场景有一定冲击力
- 16-18分：高潮场景戏剧性强，震撼有力
- 19-20分：高潮场景极其震撼，升华主题

### 维度四：伏笔意象（20分）
**评分标准**：
- 象征元素的贯穿性和巧妙性
- 意象的运用和深化程度
- 伏笔的布置和揭示节奏
- 象征意义对主题的支撑

**评分细则**：
- 0-10分：伏笔意象不明显或混乱
- 11-15分：伏笔意象有一定安排但不够巧妙
- 16-18分：伏笔意象巧妙，贯穿全文
- 19-20分：伏笔意象极其巧妙，深化主题

### 维度五：对比效果（20分）
**评分标准**：
- 文化背景的独特性和视角差异性
- 历史与现代的对比效果
- 人物命运的对比和反思
- 对比对故事深度的提升

**评分细则**：
- 0-10分：对比效果不明显或肤浅
- 11-15分：对比效果有一定展现
- 16-18分：对比效果明显，提升故事深度
- 19-20分：对比效果极其深刻，深化主题

## 评级标准

### 五星（90-100分）：可直接出版
- 内心挣扎极其深刻，贯穿整个故事
- 感情线极其深刻，推动情节发展
- 高潮场景极其震撼，升华主题
- 伏笔意象极其巧妙，深化主题
- 对比效果极其深刻，深化主题
- **总体评价**：卓越作品，可直接出版

### 四星（80-89分）：优秀作品，需少量修改
- 内心挣扎深刻，有层次感
- 感情线合理且有一定深度
- 高潮场景戏剧性强，震撼有力
- 伏笔意象巧妙，贯穿全文
- 对比效果明显，提升故事深度
- **总体评价**：优秀作品，只需少量修改即可出版

### 三星（70-79分）：合格作品，需较多修改
- 内心挣扎有一定展现但不够深刻
- 感情线基本合理但不深刻
- 高潮场景有一定冲击力
- 伏笔意象有一定安排但不够巧妙
- 对比效果有一定展现
- **总体评价**：合格作品，需要较多修改才能出版

### 二星（60-69分）：需要改进
- 内心挣扎不明显或肤浅
- 感情线混乱或不合理
- 高潮场景平淡或不合理
- 伏笔意象不明显或混乱
- 对比效果不明显或肤浅
- **总体评价**：需要大幅改进才能出版

### 一星（50-59分）：基本及格
- 内心挣扎有基本展现
- 感情线有基本逻辑
- 高潮场景有基本设置
- 伏笔意象有基本安排
- 对比效果有基本展现
- **总体评价**：基本及格，但需要大量改进

### 不合格（<50分）：需要重写
- 内心挣扎缺失
- 感情线缺失或不合理
- 高潮场景缺失
- 伏笔意象缺失
- 对比效果缺失
- **总体评价**：不合格，需要重写

## 评审报告格式

### 评审报告应包含：
1. **评审基本信息**
   - 评审小说名称
   - 评审人
   - 评审时间
   - 评审分数（每个维度分数和总分）

2. **优点分析**
   - 作品的亮点
   - 特别优秀的部分
   - 值得称赞的创作技巧

3. **缺点指出**
   - 存在的问题
   - 需要改进的部分
   - 不符合要求的方面

4. **改进建议**
   - 具体修改建议
   - 修改方向
   - 修改示例

5. **综合评价**
   - 总体评价
   - 评级结果
   - 是否达标

### 评审结果处理：
- **达标**（90-100分）：直接交付用户
- **不达标但可修改**（60-89分）：退回修改
- **不合格**（<50分）：退回重写

## 驳回和修改流程

### 评审不达标情况处理：
1. **退回构思人物Agent** → 人物档案需要改进
   - 内心挣扎维度评分<15分
   - 人物设定不符合矛盾标准

2. **退回构思背景Agent** → 背景构思需要改进
   - 历史背景不准确或肤浅
   - 文化对比维度评分<15分

3. **退回构思创意Agent** → 创意构思需要改进
   - 情节框架不合理
   - 高潮场景维度评分<15分

4. **退回构思小说Agent** → 小说正文需要改进
   - 小说质量不足
   - 总评分<90分

### 修改要求：
- 必须明确指出不足的具体位置
- 提供具体的修改方向和示例
- 重新执行对应Agent流程
- 再次评审直到达标

## 交付条件

### 交付必须满足以下条件：
1. 四Agent流程全部完成
2. 评审得分≥90分（五星标准）
3. 所有相关文件齐全：
   - 人物档案文件
   - 背景构思文件
   - 创意构思文件
   - 小说正文文件
   - 评审报告文件
4. 归档文件完整

### 交付动作：
1. 交付小说文件给用户
2. 在TODO清单中标记完成
3. 在项目进度表中更新状态
4. 记录交付时间和评审结果
5. 归档所有相关文件
