# 评审Agent工作指令

## 任务说明
**评审Agent需要对织田信长小说进行评审**

### 输入文件
1. **小说稿**：`织田信长小说.md`
2. **人物档案**：`织田信长人物档案.md`
3. **背景构思**：`织田信长背景构思.md`
4. **创意构思**：`织田信长创意构思.md`
5. **评审标准**：`矛盾小说奖评审标准.md`

### 评审要求
**必须严格按照矛盾小说奖评审标准评审**

### 评审维度（每个维度20分，总分100分）
1. **矛盾深度**（20分）：人物内心矛盾的真实性、层次性、影响性、艺术性
2. **文学质量**（20分）：语言艺术性、结构完整性、情节合理性、主题深刻性
3. **人物塑造**（20分）：人物真实性、复杂性、发展性、关系性
4. **情节设计**（20分）：情节冲突性、逻辑性、新颖性、完整性
5. **历史真实性**（20分）：历史背景真实性、文化细节准确性、时代特征真实性、历史逻辑合理性

### 矛盾小说奖标准
**评审达标标准**：
- 总分≥90分
- 矛盾深度≥18分
- 人物塑造≥16分
- 情节设计≥16分
- 历史真实性≥16分

### 评审时间
**预计评审时间**：2-4小时
**完成时间**：2026年3月24日

### 评审报告格式
**评审报告必须包含**：
1. 评审基本信息
2. 维度评分分析
3. 优点分析
4. 缺点指出
5. 改进建议
6. 综合评价

### 评审结果处理
**达标**（总分≥90分且矛盾深度≥18分）：
- 评级：五星
- 处理：直接交付用户
- 记录：记录评审分数

**不达标但可修改**（总分≥80分但矛盾深度＜18分）：
- 评级：四星或三星
- 处理：退回修改人物档案
- 修改要求：明确指出问题并提供修改建议

**不合格**（总分＜80分）：
- 评级：二星或一星
- 处理：退回构思小说Agent重写
- 重写要求：重新创作小说正文

## 评审Agent评审流程

### 第一步：阅读小说稿
仔细阅读织田信长小说，理解小说内容

### 第二步：对照评审标准
对照矛盾小说奖评审标准，逐项评分

### 第三步：评分分析
分析每个维度的得分，找出优势和不足

### 第四步：优缺点分析
分析小说的优点和缺点，指出需要改进的部分

### 第五步：改进建议
提供具体的改进建议，帮助修改

### 第六步：综合评价
给出综合评价，决定是否达标

### 第七步：生成评审报告
生成评审报告，记录评审结果

## 评审重点检查项

### 矛盾深度检查项
1. **人物内心矛盾的真实性**：织田信长的内心挣扎是否真实
2. **矛盾展现的层次性**：矛盾展现是否有层次感
3. **矛盾对情节的影响**：矛盾如何推动情节发展
4. **矛盾解决的艺术性**：矛盾如何被艺术性解决或未解决

### 文学质量检查项
1. **语言艺术性**：语言是否优美，表达是否准确
2. **结构完整性**：结构是否完整，章节安排是否合理
3. **情节合理性**：情节发展是否合理，是否符合逻辑
4. **主题深刻性**：主题是否深刻，思想性是否强

### 人物塑造检查项
1. **人物真实性**：织田信长形象是否真实，符合逻辑
2. **人物复杂性**：织田信长是否复杂，具有多重性格特点
3. **人物发展性**：织田信长发展是否符合逻辑，是否有成长过程
4. **人物关系性**：织田信长与其他人物关系是否合理，是否有互动性

### 情节设计检查项
1. **情节冲突性**：情节冲突是否强烈，戏剧性是否强
2. **情节逻辑性**：情节逻辑是否清晰，是否符合逻辑
3. **情节新颖性**：情节是否新颖，富有创意
4. **情节完整性**：情节是否完整，结构是否清晰

### 历史真实性检查项
1. **历史背景真实性**：战国时代历史背景是否真实，符合历史事实
2. **文化细节准确性**：战国时代文化细节是否准确，符合历史文化
3. **时代特征真实性**：战国时代时代特征是否真实，符合时代特点
4. **历史逻辑合理性**：历史逻辑是否合理，符合历史逻辑
