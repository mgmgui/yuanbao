# 七阶段入场/出场模板

用途: 为 Briefing/Research/Draft/Review/Adaptation/Delivery/Archive 提供统一模板，确保每个阶段都有可审计门禁。

## 统一填写项

- stage:
- owner(R):
- accountable(A):
- planned_start:
- planned_end:
- actual_end:
- linked_task_id:

## Stage 1: Briefing

### Entry
- [ ] 选题与人物对象已确认
- [ ] 受众与平台目标明确

### Checklist
- [ ] 任务卡已生成
- [ ] 完成定义(DoD)已确认
- [ ] 风险假设已记录

### Exit
- [ ] 范围冻结
- [ ] owner 与 SLA 已确认
- [ ] 可进入 Research

## Stage 2: Research

### Entry
- [ ] Briefing 已通过门禁
- [ ] 检索任务与证据范围已下发

### Checklist
- [ ] 关键事实均有来源
- [ ] 证据已分级(官方/实践/内部/风险)
- [ ] 时间线与冲突图谱完整

### Exit
- [ ] 研究包齐全
- [ ] 关键论断可追溯
- [ ] 可进入 Draft

## Stage 3: Draft

### Entry
- [ ] Research 包已通过
- [ ] 写作框架与版本策略明确

### Checklist
- [ ] 主稿结构完整
- [ ] 事实引用可追溯
- [ ] 自检表完成

### Exit
- [ ] 主稿可评审
- [ ] 已附创作说明
- [ ] 可进入 Review

## Stage 4: Review

### Entry
- [ ] 主稿与自检表齐全
- [ ] 评审人已指派

### Checklist
- [ ] 五维评分完成
- [ ] 关键项阈值判定完成
- [ ] 如不通过，驳回包完整

### Exit
- [ ] PASS: 可进入 Adaptation
- [ ] REJECT: 回退 Draft 并给出截止

## Stage 5: Adaptation

### Entry
- [ ] Review 结果为 PASS
- [ ] 平台规则与禁用项明确

### Checklist
- [ ] 微信版完成
- [ ] 小红书版完成
- [ ] 平台差异说明完成

### Exit
- [ ] 双平台版本均通过格式检查
- [ ] 可进入 Delivery

## Stage 6: Delivery

### Entry
- [ ] 双平台版本齐全
- [ ] 发布前检查清单已创建

### Checklist
- [ ] 交付包结构完整
- [ ] 版本号一致
- [ ] 交接包已提交

### Exit
- [ ] 下游签收窗口已启动
- [ ] 可进入 Archive

## Stage 7: Archive

### Entry
- [ ] Delivery 已完成
- [ ] 迁移范围已确认

### Checklist
- [ ] 归档索引已更新
- [ ] 迁移映射日志已更新
- [ ] 复盘报告已产出

### Exit
- [ ] 历史路径可追溯
- [ ] 下一周期改进项已登记
