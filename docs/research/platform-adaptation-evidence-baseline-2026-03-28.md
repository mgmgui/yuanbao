## 2. 平台差异结论

- 微信公众号: 长文阅读导向，强调导语完整、小标题节奏、事实与引用可核验。
- 小红书: 短内容互动导向，强调开头钩子、短段落节奏、标签策略与互动收口。
- 双平台共性: 必须完成合规、原创、可读性与用户价值检查。

## 3. 适配策略（执行级）

1. 不允许同文直拷，必须有平台差异说明。
2. 先评审后改写，Review 不通过禁止进入 Adaptation。
3. 双平台任何一端未过清单，Delivery 阶段不得放行。
4. 发布包必须包含两端稿件、评审结论、清单与元数据。

## 4. 刷新节拍

- 每月: 复核平台规则中心是否有变化。
- 每季度: 回顾双平台通过率与驳回原因，更新 SOP 与清单。
