1. **农民出身**: 战国时代农民与武士的巨大阶级差距
2. **身份突破**: 从农民到天下人的历史意义
3. **社会排斥**: 贵族社会的排斥与歧视
4. **阶级焦虑**: 农民出身带来的身份焦虑

### 内心挣扎（重点描写）
1. **自卑与野心**: 出身自卑与权力渴望的矛盾
2. **忠诚与背叛**: 对织田信长的忠诚与个人野心
3. **权力与孤独**: 权力顶峰带来的孤独与不安
4. **历史评价焦虑**: 对后世评价的担忧

### 历史准确性
1. **战国时代背景**: 社会结构、政治环境准确无误
2. **历史事件顺序**: 秀吉生平重要事件准确
3. **文化细节**: 战国时代文化习俗准确
4. **人物关系**: 秀吉与织田信长、德川家康关系准确

### 文学质量
1. **语言优美**: 文字流畅，描写生动
2. **情节紧凑**: 章节安排合理，节奏适中
3. **人物鲜明**: 丰臣秀吉性格特点鲜明
4. **主题深刻**: 阶级矛盾、身份焦虑主题深刻
