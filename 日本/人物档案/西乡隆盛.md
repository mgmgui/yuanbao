# 西乡隆盛人物档案

## 基本信息
- **姓名**: 西乡隆盛（Saigō Takamori）
- **出生**: 1828年
- **逝世**: 1877年
- **身份**: 日本明治维新时期的政治家、军事家
- **称号**: "最后的武士"，维新三杰之一

## 历史地位
- **明治维新**: 推动明治维新，推翻幕府政权
- **军事领导**: 戊辰战争的军事指挥者
- **西南战争**: 领导最后的武士反抗，以身殉道
- **武士精神**: 传统武士精神的代表与坚守者

## 性格特点
1. **武士精神**: 坚守武士传统价值观
2. **忠诚矛盾**: 对新政府的忠诚与对武士道的坚守
3. **悲剧命运**: 维新功臣却成为维新政府的反对者
4. **道德坚守**: 坚守原则，不惜牺牲生命
5. **领导能力**: 军事指挥与政治组织能力

## 小说创作要点
1. **武士精神**: 展现传统武士道的坚守
2. **忠诚矛盾**: 对维新政府的矛盾态度
3. **悲剧命运**: 从维新功臣到叛军的悲剧转变
4. **时代变革**: 从封建时代到现代国家的转变
5. **牺牲精神**: 以身殉道的武士精神
