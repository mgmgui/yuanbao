# 丰臣秀吉人物档案

## 基本信息
- **姓名**: 丰臣秀吉（Toyotomi Hideyoshi）
- **出生**: 1537年（一说1536年）
- **逝世**: 1598年
- **身份**: 日本战国时代至安土桃山时代的武将、政治家
- **出身**: 农民出身，最终成为关白、太政大臣，统一日本

## 历史地位
- **统一日本**: 继织田信长之后完成日本统一
- **阶级跃升**: 从农民到天下人，打破武士阶级垄断
- **政治改革**: 实施兵农分离、刀狩令等政策
- **对外扩张**: 试图征服朝鲜（朝鲜之役）

## 性格特点
1. **出身自卑**: 农民出身带来的自卑感与强烈自尊心
2. **权力渴望**: 对权力的极度渴望，渴望超越出身
3. **务实灵活**: 善于妥协与谈判，政治手腕灵活
4. **野心勃勃**: 统一后仍试图征服朝鲜，野心不断膨胀
5. **身份焦虑**: 终未能成为征夷大将军，始终有身份焦虑

## 阶级矛盾
- **出身差距**: 战国时代农民与武士的巨大阶级差距
- **身份限制**: 即使统一日本，仍无法成为征夷大将军
- **社会变革**: 通过兵农分离等措施试图改变阶级结构
- **历史评价**: 既是阶级突破的英雄，又是野心扩张的象征

## 内心挣扎
1. **自卑与自负**: 出身自卑与权力成功的矛盾心理
2. **忠诚与背叛**: 对织田信长的忠诚与个人野心间的挣扎
3. **理想与现实**: 统一日本后的理想与现实政治的冲突
4. **权力与孤独**: 权力顶峰带来的孤独与不安
5. **历史评价焦虑**: 对后世评价的担忧与焦虑

## 小说创作要点
1. **展现阶级跃升**: 从农民到天下人的历史意义
2. **刻画内心挣扎**: 自卑、野心、焦虑的复杂心理
3. **历史准确性**: 战国时代社会背景、政治环境
4. **身份焦虑**: 未能成为征夷大将军的遗憾
5. **政治手腕**: 谈判、妥协、权术的政治智慧
6. **对外扩张**: 朝鲜之役的野心与失败

## 质量控制要点
1. **字数**: 5,000-8,000字
2. **情节结构**: 紧凑合理，历史事件安排有序
3. **人物塑造**: 丰臣秀吉复杂性格的充分展现
4. **历史准确性**: 战国时代背景准确无误
5. **主题深度**: 阶级矛盾、身份焦虑等主题的深入探讨
6. **文学质量**: 语言优美，描写生动
