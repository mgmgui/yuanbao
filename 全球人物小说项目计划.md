# 全球人物小说项目计划

## 项目目标
按照明朝人物小说项目的格式和质量标准，为多个国家创作人物短篇小说：
- 每个国家：50个人物
- 每个人物：一篇5,000-8,000字的短篇小说
- 总计：6个国家 × 50人物 = 300篇短篇小说

## 项目架构
### 子agent分工：
1. **人物塑造Agent** - 创建人物基本信息、弧光设计、关键场景
2. **小说创作Agent** - 根据人物设计创作完整短篇小说
3. **评审Agent** - 评审小说质量，提出改进意见
4. **项目管理Agent** - 协调进度，归档成果

## 人物类型分布（每个国家50人）
- 皇室/贵族：10人
- 战士/军人：10人
- 艺术家/作家：10人
- 科学家/发明家：10人
- 普通人/平民：10人

## 创作流程
### 阶段一：人物塑造（Agent 1）
输出：人物档案（含弧光设计）
标准：
- 基本信息：姓名、时代、身份
- 核心矛盾：内心的挣扎/外部挑战
- 弧光设计：起点→转折→终点
- 关键场景：3个决定性瞬间
- 主题意象：贯穿全文的象征元素

### 阶段二：小说创作（Agent 2）
输出：短篇小说完整稿
标准：
- 字数：5,000-8,000字
- 结构：开头→发展→高潮→结尾
- 情感深度：参考明朝人物小说标准
- 文化准确性：符合该国历史背景

### 阶段三：评审修改（Agent 3）
输出：评审报告 + 修改建议
标准：
- 评分系统（沿用明朝项目标准）
- 评审维度：内心挣扎、感情线、高潮场景、伏笔意象、对比效果
- 修改要求：针对薄弱环节重点改进

### 阶段四：归档管理（Agent 4）
输出：归档报告 + 项目管理
任务：
- 文件组织整理
- 进度跟踪汇报
- 质量控制复核
- 最终成果归档

## 评审标准（参考明朝项目）
### 评分维度（每个维度20分，总分100分）
1. **内心挣扎**（20分）
   - 人物内在矛盾的真实性
   - 心理变化的细腻度
   - 挣扎过程的感染力

2. **感情线**（20分）
   - 爱情/亲情/友情的深度
   - 感情发展的合理性
   - 情感共鸣的强度

3. **高潮场景**（20分）
   - 关键时刻的戏剧性
   - 场景描写的震撼力
   - 结局的冲击力

4. **伏笔意象**（20分）
   - 象征元素的贯穿性
   - 意象运用的巧妙性
   - 主题统一的完整性

5. **对比效果**（20分）
   - 与其他人物的差异性
   - 文化背景的独特性
   - 故事视角的独特性

### 评级标准
- 90-100分：五星，可直接出版
- 80-89分：四星，需少量修改
- 70-79分：三星，需较多修改
- 60-69分：二星，需大量修改
- 低于60分：不合格，需重写

## 子Agent启动策略
1. **逐步启动**：先启动Agent 1（人物塑造），完成日本人物档案
2. **验证流程**：用日本人物试点，测试整个创作流程
3. **批量扩展**：流程成熟后，启动其他国家
4. **质量优先**：严格控制质量，不合格的作品需重写

## 风险控制
1. **质量风险**：通过评审Agent严格把关
2. **进度风险**：项目管理Agent每日跟踪进度
3. **文化准确性风险**：增加历史文化背景研究环节
4. **数量风险**：300篇小说需分批完成，保证质量

## 成功指标
1. **质量**：80%以上作品达到四星评级
2. **进度**：按计划时间节点完成
3. **文化准确性**：每个作品符合历史背景
4. **多样性**：涵盖不同时代、职业、性别的人物
