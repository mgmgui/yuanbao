# yuanbao - OpenClaw定时任务监控系统

OpenClaw定时任务监控系统，包含小说创作项目监控、财经监控和国际局势监控等功能。

## 项目结构

### 定时任务配置
1. **小说创作项目监控**：每10分钟执行一次
   - 进度检查、大纲预审、创作启动、评审检查
   - 优化点：增加了大纲预审环节，减少返工

2. **财经监控**：每30分钟执行一次
   - 股票、基金、可转债、黄金、商品等金融市场监控
   - 优化点：增加了决策辅助和风险预警功能

3. **国际局势监控**：每小时执行一次
   - 美以伊战况及相关事件监控
   - 优化点：升级为中东局势综合分析，多视角信源收集

4. **定时任务执行监控**：每5分钟执行一次
   - 检查所有定时任务状态、资源使用情况
   - 优化点：全面的任务监控和问题识别

### 主要功能
- 自动化的监控和报告生成
- 智能分析和预警功能
- 定时任务执行状态监控
- 元宝频道报告推送

## 使用方法
1. 系统已通过OpenClaw cron系统配置定时任务
2. 任务会自动发送消息到元宝频道
3. 监控脚本会自动执行并生成报告
4. 报告会推送到元宝频道供审查

## 监控流程
```
📊 定时任务监控系统 (已优化)
├── 📚 小说创作项目监控 (每10分钟)
│   ├── 进度检查 → 大纲预审 → 创作启动 → 评审检查 → 报告生成 → 元宝发送
│   ├── 优化点：大纲预审环节，减少返工

│   ├── 数据收集 → 波动警报 → 决策辅助 → 趋势分析 → 报告生成 → 元宝发送

├── 🌍 国际局势监控 (每小时)
```
