# 矛盾小说评审标准（基于茅盾文学奖标准）

## 评审维度（每个维度20分，总分100分）

### 1. 思想性与艺术性统一（20分）
- 思想深度：作品需体现时代精神和社会正能量
- 艺术价值：具有文学表现力和创新性
- 创新性：叙事手法、语言表达等方面的创新

### 2. 题材与人物塑造（20分）
- 现实性：反映现实生活，关注民生
- 人物形象：塑造新时代人物形象，人物鲜明生动
- 题材平衡：兼顾不同题材、风格与创作手法

### 3. 语言与完整性（20分）
- 语言质感：语言的准确性、清晰性、流畅性、美感
- 完整性：小说结构完整，章节衔接自然
- 字数：目标字数5,000-8,000字（短篇小说标准）

### 4. 创新性与时代精神（20分）
- 叙事创新：叙事手法独特，结构新颖
- 时代体现：体现当代文化命题，反映时代变化
- 艺术感染：作品具有艺术感染力

### 5. 矛盾冲突与悬念设计（20分）
- 矛盾深刻：人物内在矛盾深刻真实
- 冲突激烈：情节冲突激烈自然
- 悬念巧妙：悬念设置巧妙吸引人
- 口语化语言：贴近时代口语风格

## 评分等级
- 90-100分：五星，可直接拿奖
- 80-89分：四星，需少量修改
- 70-79分：三星，需较多修改
- 60-69分：二星，需大量修改
- 低于60分：不合格，需重写

## 评审机制
### 评审流程：
1. **评审Agent评审**：按照以上5个维度评分
2. **发现问题**：指出不符合标准的问题
3. **驳回**：如果不符合标准，驳回并要求重写
4. **重复评审**：直到达到拿奖标准才能交付

### 评审Agent要求：
1. 严格按照茅盾文学奖评审标准评审
2. 重点关注思想性与艺术性的统一
3. 注重语言质感和人物塑造
4. 要求创新性和时代精神体现
5. 突出矛盾冲突和悬念设计

## 评审标准要点总结

### 茅盾文学奖评审要点：
1. **思想性与艺术性统一** ✅
2. **题材与人物塑造** ✅
3. **语言与完整性** ✅
4. **创新性与时代精神** ✅

### 矛盾小说附加要点：
1. **矛盾深刻** ✅
2. **悬念巧妙** ✅
3. **口语化语言** ✅

## 评审Agent工作流程

### 评审Agent职责：
1. 评审小说是否符合茅盾文学奖标准
2. 评审小说是否符合矛盾小说标准
3. 发现问题并提出具体修改建议
4. 驳回不合格作品，要求重写
5. 直到达到90分以上才交付

### 评审Agent参考标准：
1. 茅盾文学奖历届获奖作品特点
2. 矛盾小说的特点（口语化、悬念、矛盾）
3. 网络文学评审标准（2026年起生效）
